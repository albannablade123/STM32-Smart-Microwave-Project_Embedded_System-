# STM32-Smart-Microwave-Project_Embedded_System-

## About the Project
This project is a final project that is included in my Embedded System course. The project is coded in C, and utilize the STM32F103C6. The purpose of this project is to apply high level of the C language design patterns
and learn more about designing, maintaining and creating effecient Embedded System. 

## Key Features
* Power Saving
* USART Communication
* ADC 
* Interrupts

## Built With 
* C 
* Keil MDK
* Proteus
* CubeMX
